# Library Management System (Frontend Demo)

This project demonstrates the frontend user interface and workflow of a Library Management System.

## Features
- Book listing with availability status (Available / Issued)
- Dynamic book search functionality
- Clean and simple user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Live Demo
https://haleemdevop.github.io/library-management-system/

## Note
This project showcases the frontend functionality. Backend services were implemented separately and tested locally.
